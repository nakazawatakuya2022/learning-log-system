# 画面遷移（簡易）

Status: Draft  
Last Updated: 2026-02-08

[ /logs  トップ画面（登録＋一覧＋検索） ]
|
|（一覧のログをクリック）
v
[ /logs/{id}  詳細画面 ]
|
|（編集／解決追記／ナレッジ化）
v
[ /logs/{id}  詳細画面（更新後） ]
|
|（一覧に戻る）
v
[ /logs  トップ画面 ]


## 補足説明

- 画面は **2画面構成**（/logs、/logs/{id}）とする
- 登録・一覧・検索は **トップ画面に統合**する
- 詳細画面では
    - 内容編集
    - 解決方法追記
    - 解決状態／ナレッジの切り替え
        
        を行う
        
- 編集後は
    - 詳細画面に留まる
    - またはトップ画面へ戻る
        
        のどちらも可能とする
