# スコープ定義（Scope）

Status: Draft  
Last Updated: 2026-02-08

---

## 1. MVPで対応する範囲
- 学習ログの登録
- 学習ログの一覧表示・検索
- 学習ログの詳細表示
- 学習ログの編集
- 解決状態・ナレッジ化の管理

---

## 2. MVPで対応しない範囲（非スコープ）
- ユーザー認証
- ログ削除機能
- 投稿者管理・権限管理
- 全文検索
- タグの正規化
- 変更履歴管理
- Q&A／回答機能
- ベストアンサー機能

---

## 3. 補足
本プロジェクトは MVP として最小構成を優先し、
将来的な拡張を前提とした設計とする。
